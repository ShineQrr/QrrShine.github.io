typeof和instanceof的区别
发表于2020-08-05|前端
|字数总计:304|阅读时长:1分钟

typeof

返回一个字符串，会返回undefined、boolean、string、number、object、function、symbol用于检测变量的数据类型，表示未经计算的操作数的类型。

instanceof

返回布尔值，A instanceof B, 判断是否A是B的实例，即B的prototype属性是否在实例A的原型链上。检测构造函数的prototype属性是否出现在某个对象实例的原型链上。
